# Python Test Practice 01

Pequeno teste prático desenvolvido após a conclusão do meu curso de Python.

## Conteúdos aplicados

- Estruturas condicionais (if, elif, else)
- Operadores matemáticos
- Listas e loops (for)
- Lógica para encontrar maior valor manualmente
- Funções com múltiplos retornos
- Simulação simples de controle de lucro diário

## Exercícios

1. Validação de número (negativo, par ou ímpar)
2. Manipulação de listas (filtro, soma e maior valor)
3. Sistema de notas com cálculo de média e status
4. Simulação de caixa diário (inspirado na minha rotina de trabalho)

Prática focada em reforçar fundamentos e escrever código organizado.
